# NYC Path Finder Web App using Go

A Go-based web application that takes a pair of coordinates within New York City and returns an interactive Leaflet map displaying the best path between the two points.

The project is a Go implementation of Dijkstra’s algorithm for finding the shortest path between nodes in a graph. As a data scientist with a background in Python, I created this project to learn Go’s approach to data structures, file handling, and algorithm implementation.

This exercise provides practical insight into Go’s type system, error handling, and standard library features.


## Features

- Accepts start and end coordinates within NYC
- Calculates the optimal path between the points
- Renders the route on an interactive Leaflet map
- Simple, fast, and easy-to-use web interface
